# ykit-starter-antd
ykit ant-design 示例工程

## Getting Started
```
$ git clone https://github.com/roscoe054/ykit-starter-antd.git
$ cd ykit-starter-antd
$ npm install 或者 yarn install
```

## Run Dev
```
$ npm run dev
```
访问 http://127.0.0.1:8080/ykit-starter-antd/index.html

## Build
```
$ ykit pack -m
```
